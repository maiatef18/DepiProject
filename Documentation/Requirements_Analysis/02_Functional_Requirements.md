# Functional Requirements

## 1. Authentication
*   Users must be able to register as Patient or Hospital.
*   Secure Login/Logout with Token Management.

## 2. Hospital Module
*   Add/Update/Delete medical services (e.g., ICU, MRI, Dental).
*   Upload profile images.
*   View dashboard statistics (Review count, Average rating).

## 3. Patient Module
*   Update profile and upload profile picture.
*   Search services by Category (e.g., Emergency, Lab) or Keyword.
*   Save services to a "Wishlist".
*   Add reviews and ratings to utilized services.

## 4. Core Features
*   **Comparison Engine:** Compare two services based on Price, Rating, and Distance.
*   **Geo-Location:** Filter hospitals based on user coordinates.
