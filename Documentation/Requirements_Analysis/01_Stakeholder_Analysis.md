# Stakeholder Analysis

**Project Name:** Mos3ef

| Stakeholder | Role | Key Needs/Interests |
| :--- | :--- | :--- |
| **Patients** | End User | Fast search, accurate pricing, trusted reviews, proximity-based results. |
| **Hospitals** | Service Provider | Ability to update service availability, view user feedback, analyze profile stats. |
| **Administrators** | System Owner | User management, role assignment, system integrity oversight. |
